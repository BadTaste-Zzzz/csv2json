CSV 2 JSON

本地转换，不上传数据

把问答表格转成训练 JSON

上传 CSV、XLSX 或直接粘贴两列表格，字段使用 user 和 assistant，一键生成可下载的 JSONL 或 JSON。

Input
user,assistant  |  你好,你好！有什么可以帮你？
Map
user -> role:user   assistant -> role:assistant
Output
{"messages":[{"role":"user"},{"role":"assistant"}]}

输入

表头必须包含问题列和回答列。

输出格式
等待输入。

输出

空白问答行会自动跳过。

0 有效行
0 字段数
0 跳过行
暂无预览
